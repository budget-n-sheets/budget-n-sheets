Basics

The pages

Summary This page displays the income and expenses total for each month, and an year overall as well.

Jan, Feb, ..., Dec These pages are where you enter statements, like expenses, income and transactions, with cash and (bank) account balance. These pages are referred to as month page.

Cards This page is where you enter statements, like expenses and refunds, with a (credit) card.

Cash Flow This page shows your combined account balance per day after a cash flow.

Cash Flow Here you can create custom tags to tag your statements.

Decimal separator

Your spreadsheet uses () as decimal separator.

Plus and minus signs

Use a positive number whenever a value is towards your balance.
Examples: income, deposits, transfers from someone and refunds.

Use a negative number, with a - (minus) sign as prefix, whenever the value is outwards your balance.
Examples: expenses, withdrawals and transfers to someone.

Protected sheets and ranges

Do not edit protected sheets and ranges if it is unnecessary! Editing protected sheets and ranges may break the spreadsheet's functionality, and the add-on may not work properly. Updates to the spreadsheet may come to a failure as well.

The following actions are safe despite showing a warning: show/hide tabs, expand/collapse column and row groups, show/hide columns and rows.

Entering statements

expand_less

For transactions with cash, enter the statements in the Wallet table.

For transactions with account balance (like using debit card), enter the statements in the respective account table.

For transactions with credit card, enter the statements in the Cards page.

Register income

To mark a statement as income, tag it with the #rct code. If the statement is in an account table, then it also needs a deposit or transfer tag.

Ignore a value

The value of a statement can be ignored and not accounted in the sum of expenses by tagging the statement with the #ign code.

Account transactions

Account statements of deposits, transfers and withdrawals (transactions) requires a built-in tag:

  • Deposits: #dp, only positive values
  • Transfers: #trf
  • Withdrawals: #wd, only negative values

A summary of transactions is displayed in the header of each account table.

Accounts and Cards

expand_less

Set account balance

You can set an initial month and balance for each account. From the initial month, the account balance will be affect by the statements.

  1. Click Add-ons > Budget n Sheets > Open Accounts & Cards.
  2. Select the account_balance Accounts tab.
  3. Click Edit in one of the accounts.
  4. Edit name, initial month and initial balance.
  5. Click Save.

Add cards

You can create a Card for each credit card you own. The code for that Card is then used to associate the respective statements with your credit card in the Cards page.

  1. Click Add-ons > Budget n Sheets > Open Accounts & Cards.
  2. Select the credit_card Cards tab.
  3. Click add Add card.
  4. Enter the name, code, code aliases and credit limit.
  5. Click Add.

Custom tags

expand_less

Use tags to track specific statements and generate analytics by tag and category.

Create a tag

  1. Click the Tags page.
  2. Create one tag per row. The options category and code are required.

Options

  • category: Category of the tag.
  • analytics: If checked, the tag is included in the Stats for Tags report from the Cool Gallery.
  • code: The code of the tag. Charset accepted: 0-9, a-z, A-Z, _

Using tags

Use a tag by entering # followed by the tag's code in the tag column. You can combine multiple tags, and sepparation with space or comma is optional.

Subsubcategory

A tag can have subtags of its own by appending a sufix to a main tag. For example, for two different trips, you can create the tags #trip, #trip1 and #trip2. For each trip, you tag the expenses with tags #trip1 and #trip2. Meanwhile, the tag #trip is accounting the sum of both trips, since it uses the same initial code: trip.
